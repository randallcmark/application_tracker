# Archived Planning Notes

This directory records the archive policy for older planning artifacts.

Current planning starts with the three hub documents:

1. Vision: `docs/PRODUCT_VISION.md`
2. Strategy and order: `docs/roadmap/implementation-sequencing.md`
3. Task breakdown: `docs/roadmap/task-map.md`

Older roadmap, strategy, analysis, and design-refresh documents may remain in their original
locations when they are still useful as background or supporting references. Those files should
start with a historical or supporting-context banner that points back to the three current hubs.

Do not treat archived or superseded planning notes as the current execution plan unless the task
map explicitly routes to them as supporting detail for a workstream.
